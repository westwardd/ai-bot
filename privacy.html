Privacy Policy

Last updated: August 2025

This application is a Google Workspace Add-on that automates email and spreadsheet processing to improve user productivity. We value your privacy and are committed to protecting your personal data.

1. Data Collection and Use

Our application accesses Google user data solely for the functionality described and only with the user’s explicit consent through OAuth 2.0. Specifically, the app may request access to the following:

  • Gmail content and metadata (gmail.readonly, gmail.modify, gmail.send) — used to read, reply, and send messages based on defined triggers.
  • Google Sheets (spreadsheets) — used to create and update sheets based on incoming emails.
  • External requests (script.external_request) — used to send notifications or access third-party APIs as needed.
  • Script triggers (script.scriptapp) — used to automate the application in the background at time intervals.

We do not store, share, or sell any user data. All processing happens within the Google Apps Script environment provided by Google.

2. Data Protection Measures

  • The app runs inside Google’s secure infrastructure (Google Apps Script).
  • No user data is stored on external servers or transmitted outside Google services.
  • All access to user data is restricted to the minimum necessary scopes and only used while the user is actively using the add-on.
  • All OAuth tokens are managed by Google’s secure systems.

3. User Control and Revocation

Users can revoke the app’s access to their Google account at any time via their Google account permissions page. Once revoked, the app will no longer access any data.
