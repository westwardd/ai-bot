# Gmail AI Auto-Reply Bot This is a Google Apps Script project that automatically reads incoming Gmail messages and sends intelligent replies using AI. ## ✨ Features - 📬 Reads incoming emails - 🤖 Sends smart, automatic replies - 🔒 Runs securely within your Google account - 🧠 Uses OpenAI API for generating context-aware responses ## 🔐 Permissions This app uses the following permissions: - Read Gmail inbox (to detect new messages) - Send Gmail messages (to reply) - Access user email (to identify the user account) We **do not** store or share your personal data. All processing happens within your account via Google Apps Script. ## 📄 Privacy Policy We value your privacy. Read our [Privacy Policy] https://github.com/westwardd/ai-bot/blob/master/privacy.txt to learn more. ## 🏠 Homepage This page serves as the homepage for the Gmail AI Auto-Reply Bot.
